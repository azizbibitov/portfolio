Features

What it includes

A complete messenger UI concept covering every major screen a real messaging app would have.

💬

Chat List & Room

Chat list with stories strip at the top, per-conversation message bubbles with sent/received states, keyboard handling, and a custom text input view with emoji support.

📖

Stories

Full-screen story viewer with a segmented progress bar that auto-advances through slides. Stories are accessible from the horizontal strip on the chat list.

🗞️

Social Feed

Paged feed with posts — profile pictures, like/comment/share/bookmark actions, and a search bar. Supports multiple feed tabs via a paging view controller.

🎵

Music Player

Dedicated music tab with a track list and a custom player UI. Integrated as a full tab alongside Chat, Feed, Services, and Profile.

🔐

Pin Code Lock

App lock with pin code entry and confirmation screens. Uses UserDefaults for persistence. Custom dot-based pin indicator with shake animation on wrong input.

🎨

Appearance & Themes

Appearance screen lets users pick custom UI accent colors from a collection view of color swatches. Dark mode fully supported across all screens via named color assets.

Technology

Tech Stack

Pure UIKit with EasyPeasy for Auto Layout, named color assets for dark mode, and MVC architecture throughout.
