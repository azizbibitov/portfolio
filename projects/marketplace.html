Features

What it does

An end-to-end marketplace: buyers shop and pay on iPhone, admins manage the catalogue and fulfil orders on Mac, and a single REST API ties it all together.

🛍️

Browse & Shop

Product grid organised by category with live stock badges (in stock, only-N-left, out of stock), search, and a swipeable image carousel on the detail screen.

💳

Razorpay Payments

The server creates Razorpay orders in paise, verifies an HMAC-SHA256 signature on completion, handles raw-body webhooks, and decrements stock once a payment succeeds.

🔐

Secure Auth

JWT sessions with bcrypt-hashed passwords and role-based access (buyer vs admin). Tokens persist across launches in the iOS Keychain for seamless re-entry.

🧾

Transaction History

Orders grouped by month with a full status lifecycle — pending → paid → shipped → delivered — plus refunded and cancelled states, colour-coded throughout.

🖥️

macOS Admin App

A native SwiftUI desktop app to run the store: add, edit, and delete products, upload images, and fulfil orders with one-click ship and refund actions.

☁️

Cloud Image Hosting

Product photos are uploaded as multipart form data, streamed straight to Cloudinary from an in-memory buffer, and the returned URLs are stored on the product.

Technology

Tech Stack

Two native SwiftUI clients backed by a clean three-tier Node.js API, with Razorpay, Cloudinary, and a Jest test suite.
