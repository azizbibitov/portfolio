Features

What it does

A courier delivery iOS app — place orders, track parcels in real time, select city and address, and manage delivery history.

📦

Place an Order

Multi-step order flow: select sender and recipient city, enter addresses, choose parcel type and weight, and confirm.

🚚

Real-Time Tracking

Live parcel status updates from pickup to delivery with a timeline view of each stage.

📍

City Selection

Searchable city picker with a clean list UI for selecting origin and destination from all supported delivery cities.

🔐

OTP Authentication

Phone number registration and login via one-time SMS code with a custom OTP input field.

📜

Order History

Full list of past and active orders with detailed status view and the ability to repeat a previous delivery.

🌙

Dark Mode

Full adaptive theme with named color assets for seamless dark and light appearance switching.
